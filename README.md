# Try Deno

Test [Deno](https://deno.com/) features.

### Checklist

- Monorepo setup
  - ~~Centralized control (execute commands from the root of the project)~~ (in Deno 2.1.0)
  - Shared local packages
- React SPA
- Tests
- Use [JSR](https://jsr.io/) package

### Get started

- Run in dev mode

```sh
$ deno task dev
```

- Run in prod mode

```sh
$ deno task start
```

- Navigate to `http://localhost:3001/`
